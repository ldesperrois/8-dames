Le programme « huit_reines_with_pdf.py » permet en plus la création d’un PDF récapitulatif de toutes les solutions possibles, mais il n’est pas possible, du moins pour nous étudiant, de l’exécuter sur les PC de l’IUT, car il nécessite le module « FPDF ». 

Dans le dossier "Exemple PDF" se trouvent Le fichier «solution_6_dame.pdf » qui est un exemple de fichier, générer par ce programme pour le cas ou n = 6, et le fichier “solution_8_dame.pdf” pour le cas ou n=8.
